# i18n Package

Internationalization package for the OpenAI monorepo.

## Usage

### Extract translation keys

```bash
pnpm extract
```

### Watch mode (extract on file changes)

```bash
pnpm extract:watch
```

## Namespaces

- `common`: Shared translations across all packages
- `client`: Client package translations
- `admin`: Admin package translations
- `api`: API package translations
